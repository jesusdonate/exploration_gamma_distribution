Gamma Distribution

Assignment from STAT 520 - Statistical Inference.

Gamma PDF

\[ f(x;\alpha,\beta)=\frac{1}{\Gamma(\alpha)\beta^\alpha}x^{\alpha-1}e^{-x/\beta},\quad x>0 \]

Parameters

\(\alpha\) (alpha): this parameter is know as the shape parameter because \(\alpha\) controls the overall shape of the distribution rather than simply shifting or stretching/shrinking it.

\(\beta\) (beta): this parameter is called the scale parameter because \(\beta\) changes the distribution by stretching or compressing its shape. The larger the scale parameter, the more spread out the distribution.

Support

x, \(\alpha\), and \(\beta\) must be positive real numbers.

Use the interactive page to tune parameters and inspect area under the curve.

Moments

Expected Value

\(\mathbb{E}[X]=\alpha\beta\)

Variance

\(\mathrm{Var}(X)=\alpha\beta^2\)

Derivation Details

Expected Value

\[E(X)=\int_0^\infty x\cdot\frac{1}{\Gamma(\alpha)\beta^\alpha}x^{\alpha-1}e^{-x/\beta}\,dx\]

\[=\frac{1}{\Gamma(\alpha)\beta^\alpha}\int_0^\infty x^\alpha e^{-x/\beta}\,dx\]

\[=\frac{\Gamma(\alpha+1)\beta^{\alpha+1}}{\Gamma(\alpha)\beta^\alpha}\int_0^\infty\frac{1}{\Gamma(\alpha+1)\beta^{\alpha+1}}x^\alpha e^{-x/\beta}\,dx\]

\[=\frac{\alpha\Gamma(\alpha)\beta}{\Gamma(\alpha)}\cdot1\]

\[E(X)=\alpha\beta\]

Variance

\[E(X^2)=\int_0^\infty x^2\cdot\frac{1}{\Gamma(\alpha)\beta^\alpha}x^{\alpha-1}e^{-x/\beta}\,dx\]

\[=\frac{1}{\Gamma(\alpha)\beta^\alpha}\int_0^\infty x^{\alpha+1}e^{-x/\beta}\,dx\]

\[=\frac{\Gamma(\alpha+2)\beta^{\alpha+2}}{\Gamma(\alpha)\beta^\alpha}\int_0^\infty\frac{1}{\Gamma(\alpha+2)\beta^{\alpha+2}}x^\alpha e^{-x/\beta}\,dx\]

\[=\frac{\Gamma(\alpha+2)\beta^{\alpha+2}}{\Gamma(\alpha)\beta^\alpha}\cdot1\]

\[=(\alpha+1)\alpha\beta^2=\alpha^2\beta^2+\alpha\beta^2\]

\[V(X)=E(X^2)-[E(X)]^2\]

\[=\alpha^2\beta^2+\alpha\beta^2-(\alpha\beta)^2\]

\[V(X)=\alpha\beta^2\]

Transformations

Univariate or bivariate transformations from the gamma distribution can generate additional random variables. This section is set up for three transformations.

Transformation 1: Gamma to Chi-Square \(\frac{2X}{\beta} \sim \chi^2_{2\alpha}\)

Recall:

\[ \frac{2X}{\beta} \sim \chi^2_{2\alpha}, \quad \text{where } X \sim \Gamma(\alpha,\beta) \]

Let \[ Y = \frac{2X}{\beta} \]

Then \[ x = \frac{\beta y}{2} \] and \[ \frac{dx}{dy} = \frac{\beta}{2} \]

New support: \[ 0 < x < \infty \implies 0 < \frac{\beta y}{2} < \infty \implies 0 < y < \infty \]

Using the transformation formula:

\[ \begin{align*} f_Y(y) &= f_X\left(\frac{\beta y}{2}\right) \left|\frac{dx}{dy}\right|, \qquad 0 < y < \infty \\[6pt] &= \frac{1}{\Gamma(\alpha)\beta^\alpha} \left(\frac{\beta y}{2}\right)^{\alpha-1} e^{-\frac{\beta y}{2\beta}} \cdot \frac{\beta}{2} \\[6pt] &= \frac{1}{\Gamma(\alpha)\beta^\alpha} \cdot \frac{\beta^{\alpha-1} y^{\alpha-1}}{2^{\alpha-1}} \cdot e^{-y/2} \cdot \frac{\beta}{2} \\[6pt] &= \frac{1}{\Gamma(\alpha)\beta^\alpha} \cdot \frac{\beta^\alpha y^{\alpha-1}}{2^\alpha} \cdot e^{-y/2} \\[6pt] &= \frac{1}{\Gamma(\alpha)2^\alpha} y^{\alpha-1} e^{-y/2} \\[6pt] &= \frac{1}{\Gamma\left(\frac{2\alpha}{2}\right) 2^{2\alpha/2}} y^{\frac{2\alpha}{2} - 1} e^{-y/2}, \qquad 0 < y < \infty \end{align*} \]

This is the pdf of:

\[ \chi^2_{2\alpha} \]

Thus, \[ \boxed{Y = \frac{2X}{\beta} \sim \chi^2_{2\alpha}} \]

Transformation 2: Gamma to Beta \(\frac{X_1}{X_1+X_2} \sim \operatorname{Beta}(\alpha,\beta)\)

Recall: \(X_1 \sim \Gamma(\alpha,1)\), \(X_2 \sim \Gamma(\beta,1)\), and \(X_1 \perp X_2\).

\[ \frac{X_1}{X_1+X_2} \sim \operatorname{Beta}(\alpha,\beta) \]

Joint PDF:

\[ f_{X_1,X_2}(x_1,x_2) = \frac{1}{\Gamma(\alpha)\Gamma(\beta)} x_1^{\alpha-1} x_2^{\beta-1} e^{-(x_1+x_2)}, \qquad x_1>0,\; x_2>0 \]

Let \(Y_1 = X_1 + X_2\) and \(Y_2 = \frac{X_1}{X_1+X_2}\).

Then the inverse transformation is

\[ x_1 = y_1 y_2, \qquad x_2 = y_1(1-y_2) \]

The Jacobian:

\[ J = \begin{vmatrix} y_2 & y_1 \\ 1-y_2 & -y_1 \end{vmatrix} = -y_1 \quad\Rightarrow\quad |J| = y_1 \]

Support: \(0 < y_1 < \infty,\; 0 < y_2 < 1\).

\[ \begin{align*} f_{Y_1,Y_2}(y_1,y_2) &= f_{X_1,X_2}(y_1y_2,\; y_1(1-y_2)) \cdot |J| \\[6pt] &= \frac{1}{\Gamma(\alpha)\Gamma(\beta)} (y_1y_2)^{\alpha-1} (y_1(1-y_2))^{\beta-1} e^{-(y_1y_2 + y_1(1-y_2))} \cdot y_1 \\[6pt] &= \frac{1}{\Gamma(\alpha)\Gamma(\beta)} y_1^{\alpha-1}y_2^{\alpha-1} y_1^{\beta-1}(1-y_2)^{\beta-1} e^{-y_1} \cdot y_1 \\[6pt] &= \frac{1}{\Gamma(\alpha)\Gamma(\beta)} y_1^{\alpha+\beta-1} y_2^{\alpha-1} (1-y_2)^{\beta-1} e^{-y_1} \end{align*} \]

\[ \begin{align*} f_{Y_2}(y_2) &= \int_0^\infty \frac{1}{\Gamma(\alpha)\Gamma(\beta)} y_1^{\alpha+\beta-1} y_2^{\alpha-1} (1-y_2)^{\beta-1} e^{-y_1} \, dy_1 \\[6pt] &= \frac{1}{\Gamma(\alpha)\Gamma(\beta)} y_2^{\alpha-1}(1-y_2)^{\beta-1} \int_0^\infty y_1^{\alpha+\beta-1} e^{-y_1} \, dy_1 \\[6pt] &= \frac{1}{\Gamma(\alpha)\Gamma(\beta)} y_2^{\alpha-1}(1-y_2)^{\beta-1} \Gamma(\alpha+\beta) \end{align*} \]

\[ f_{Y_2}(y_2) = \frac{\Gamma(\alpha+\beta)}{\Gamma(\alpha)\Gamma(\beta)} y_2^{\alpha-1}(1-y_2)^{\beta-1}, \qquad 0 < y_2 < 1 \]

This is the PDF of a \(\operatorname{Beta}(\alpha,\beta)\) distribution.

\[ \boxed{Y_2=\frac{X_1}{X_1+X_2} \sim \operatorname{Beta}(\alpha,\beta)} \]

Transformation 3: Gamma to Gamma with beta parameter 1 \(\frac{X}{\beta} \sim \Gamma(\alpha, 1)\)

Let \(X \sim \Gamma(\alpha, \beta)\) and define

\[ Y = \frac{X}{\beta}. \]

\[ x = \beta y, \qquad \frac{dx}{dy} = \beta \]

\[ 0 < x < \infty \;\implies\; 0 < \beta y < \infty \;\implies\; 0 < y < \infty \]

\[ \begin{align*} f_Y(y) &= f_X(\beta y) \left|\frac{dx}{dy}\right|, \qquad 0 < y < \infty \\[6pt] &= \frac{1}{\Gamma(\alpha)\beta^\alpha} (\beta y)^{\alpha-1} e^{-\beta y / \beta} \cdot \beta \\[6pt] &= \frac{1}{\Gamma(\alpha)\beta^\alpha} \beta^{\alpha-1} y^{\alpha-1} e^{-y} \cdot \beta \\[6pt] &= \frac{1}{\Gamma(\alpha)\beta^\alpha} \beta^\alpha y^{\alpha-1} e^{-y} \\[6pt] &= \frac{1}{\Gamma(\alpha)} y^{\alpha-1} e^{-y}, \qquad 0 < y < \infty \end{align*} \]

Thus,

\[ \boxed{Y = \frac{X}{\beta} \sim \Gamma(\alpha, 1)} \]
